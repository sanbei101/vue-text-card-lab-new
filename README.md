# Vue 文字卡片实验室

一个配置驱动的 Vue 3 文字卡片生成器。

## 已实现

- 12 套真正不同的卡片模板
- SVG 原生绘制，不依赖截图插件
- 根据模板宽度动态测量文字
- 二分查找最大可用字号
- 动态规划选择更自然的中文断行
- 自动识别高亮关键词
- 高亮位置随文字行布局变化
- 基于标题的确定性装饰布局
- 导出 1800 × 2400 PNG
- 移动端响应式界面

## 运行

```bash
npm install
npm run dev
```

构建：

```bash
npm run build
```

## 核心文件

```text
src/
├─ App.vue
├─ card-engine.ts       # 测量、断行、字号、自适应布局
├─ templates.ts         # 12 套模板配置
├─ types.ts
└─ components/
   ├─ CardSvg.vue       # SVG 模板渲染器
   └─ CardTile.vue
```

## 为什么选择 SVG

SVG 的渲染模型更接近 Android Canvas：

- 背景、文字、路径和贴纸都是绘制指令
- 不依赖 DOM 截图还原
- 导出清晰
- 每套模板可以拥有完全不同的绘制逻辑
- 以后迁移到 Kotlin Canvas 时，结构很好映射

## 继续产品化时建议补充

1. 将字体文件作为应用资源加载，保证不同设备一致。
2. 将模板配置和装饰素材由服务端下发。
3. 加入真正的中文分词与关键词提取。
4. 为超长内容提供分页模板。
5. 加入副标题、作者、日期、图片槽位。
6. 将模板渲染器拆成独立的 layer 系统。
7. 使用 OffscreenCanvas / Web Worker 并行生成大量预览。
